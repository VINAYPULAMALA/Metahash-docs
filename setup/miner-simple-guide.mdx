# Miner Simple Guide

This is the fastest path to bring a MetaHash miner online without diving into every configuration detail. Follow the steps below in order and you will be ready to bid in auctions within minutes.

## Simple Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/fx-integral/metahash.git
    cd metahash
    ```
  </Step>
  <Step title="Create a Python environment">
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    pip install -U pip wheel uv
    uv pip install -e .
    ```
  </Step>
  <Step title="Prepare environment variables">
    ```bash
    cp .env.template .env  # if the template exists
    ```
    Edit `.env` with your `WALLET_PASSWORD` (and adjust `BITTENSOR_NETWORK` if you need a network other than `finney`). If no template is available, create the file manually before running the miner.
  </Step>
</Steps>

## Wallet Setup

```bash
# create coldkey
btcli wallet new_coldkey --wallet.name metahash

# create hotkey
btcli wallet new_hotkey --wallet.name metahash --wallet.hotkey minerx

# register hotkey on subnet 73
btcli register --netuid 73 --wallet.name metahash --wallet.hotkey minerx
```

## Run the Miner

```bash
python neurons/miner.py \
  --netuid 73 \
  --wallet.name metahash \
  --wallet.hotkey minerx \
  --subtensor.network finney \
  --miner.bids.netuids 62 \
  --miner.bids.amounts 20.0 \
  --miner.bids.discounts 500 \
  --axon.port 8091 \
  --axon.external_port 8091 \
  --logging.debug \
  --payment.validators SOURCE_STAKE_HOTKEY
```

<Warning>
`--payment.validators` must point to a hotkey with α stake (your own or a validator hotkey). If the hotkey lacks stake, your α payments will fail and bids will never clear.
</Warning>

<Warning>
`--miner.bids.discounts` is expressed in basis points (bps), not percent. Examples: `500` = 5%, `700` = 7%, `900` = 9%.
</Warning>

<Note>
Keep `--axon.port` and `--axon.external_port` aligned so validators can reach your miner for auction participation.
</Note>
