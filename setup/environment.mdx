# Environment Setup

Install the required software and dependencies for running a MetaHash miner.

## Installation Steps

Run the following commands to set up your environment:

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y python3 python3-venv python3-pip git
git clone https://github.com/fx-integral/metahash.git
cd metahash
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip wheel uv
uv pip install -e .
pip install -U bittensor-cli
```

## Step-by-Step Breakdown

<Steps>
  <Step title="Update System Packages">
    ```bash
    sudo apt update && sudo apt upgrade -y
    ```
    Updates your Ubuntu system to the latest packages.
  </Step>

  <Step title="Install Python and Git">
    ```bash
    sudo apt install -y python3 python3-venv python3-pip git
    ```
    Installs Python 3, virtual environment tools, pip, and git.
  </Step>

  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/fx-integral/metahash.git
    cd metahash
    ```
    Downloads the MetaHash repository and navigates into it.
  </Step>

  <Step title="Create Virtual Environment">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```
    Creates and activates a Python virtual environment.
  </Step>

  <Step title="Install Dependencies">
    ```bash
    pip install -U pip wheel uv
    uv pip install -e .
    pip install -U bittensor-cli
    ```
    Installs required Python packages and the Bittensor CLI.
  </Step>
</Steps>

## Common Issues

<AccordionGroup>
  <Accordion title="Command 'python' not found">
    **Cause**: Python is not aliased to `python3`

    **Fix**: Always use `python3` instead of `python` in commands
  </Accordion>

  <Accordion title="async-substrate-interface mismatch">
    **Cause**: MetaHash requires version 1.3.1

    **Fix**: Reinstall specific version:
    ```bash
    pip install async-substrate-interface==1.3.1
    ```
  </Accordion>

  <Accordion title="Missing .env.template">
    **Cause**: Repository structure changed

    **Fix**: Create `.env` manually (see [Configuration](/setup/configuration))
  </Accordion>
</AccordionGroup>

## Verification

After installation, verify your setup:

```bash
python3 --version  # Should show Python 3.8+
btcli --version    # Should show Bittensor CLI version
```
