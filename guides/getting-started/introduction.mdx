# MetaHash Miner Setup Guide

This comprehensive guide will walk you through setting up a MetaHash miner on Subnet 73.

## Overview

- **Network**: Finney (testnet)
- **Platform**: Ubuntu 22.04 / AWS EC2
- **Setup Time**: ~40 minutes
- **Subnet ID**: 73

## What You'll Need

- Ubuntu 22.04 system (local or AWS EC2)
- Basic command line knowledge
- At least 0.15 TAO for staking
- Open port 8091 (TCP)

## Setup Steps

Follow these pages in order:

<Steps>
  <Step title="Environment Setup">
    Install Python, dependencies, and clone the repository
  </Step>
  <Step title="Wallet & Hotkey Creation">
    Create your Bittensor wallet and register on Subnet 73
  </Step>
  <Step title="Environment Configuration">
    Set up your .env file with credentials
  </Step>
  <Step title="Staking">
    Stake TAO for α (alpha) source
  </Step>
  <Step title="Running the Miner">
    Launch your miner with the correct parameters
  </Step>
  <Step title="Port Setup">
    Configure AWS security groups and verify port access
  </Step>
</Steps>

## Expected Outcome

After completing this guide, you'll have:

- ✅ A fully configured MetaHash miner
- ✅ Registered on Subnet 73 with a UID
- ✅ Active bidding with α tokens
- ✅ Proper monitoring and logging
- ✅ Automatic restart capability
