# Welcome to MetaHash

MetaHash is a subnet on the Bittensor network (Subnet 73) running on the Finney testnet. This documentation provides comprehensive setup instructions, troubleshooting guides, and best practices for running a MetaHash miner.

## What is MetaHash?

MetaHash (Subnet 73) is part of the Bittensor ecosystem, enabling decentralized mining operations with Alpha (α) token bidding mechanisms.
